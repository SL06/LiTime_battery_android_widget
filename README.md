This Android app is based on ChadJ/litime-bluehooth battery app, https://github.com/chadj/litime-bluetooth-battery. Thanks a lot for your work! I converted is HTML code core to an Android app using AI.  Thus, I don’t deserve any credit, haha! The purpose of this app is to create a widget for my Android TV box that monitors the battery state every hour. The battery is continuously plugged into its charger, but I noticed it discharged without any apparent reason the other day. Unplugging and replugging the charger restarted the charging process. Now, with the widget, I can easily monitor the battery’s state.
