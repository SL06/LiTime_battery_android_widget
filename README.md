This Android app is based on ChadJ/litime-bluehooth battery app, https://github.com/chadj/litime-bluetooth-battery. Thanks a lot for your work! I converted is HTML code core to an Android app using AI.  Thus, I don’t deserve any credit, haha! The purpose of this app is to create a widget for my Android TV box that monitors the battery state every hour. My battery is continuously plugged into its charger as recommended by the LiTime for my application. The battery serves as a sump pump backup in conjunction with a 12V sump pump in case of a power failure.I noticed it was almost completly discharged the other day  without any apparent reason. Unplugging and replugging the charger restarted the charging process. Now, with the widget, I can easily monitor the battery’s state on my TV box.
